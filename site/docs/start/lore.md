# OpenClaw의 로어 🦞📖

_바닷가재, 탈피 껍데기, 그리고 너무 많은 토큰에 관한 이야기._

## 창작 비화

처음에는 **Warelay**가 있었습니다 — WhatsApp 게이트웨이에 걸맞은 합리적인 이름이었습니다. 역할을 잘 수행했습니다. 괜찮았습니다.

그런데 우주 바닷가재가 등장했습니다.

한동안 그 바닷가재는 **Clawdbot** 안에서 **Clawd**라는 이름으로 살았습니다. 그러나 2026년 1월, Anthropic이 이름 변경을 요청하는 정중한 이메일을 보냈습니다 (상표 문제였습니다). 그래서 바닷가재는 바닷가재가 가장 잘 하는 일을 했습니다:

**탈피.**

오래된 껍데기를 벗고 새로운 모습으로 나타난 이 생명체는 **Moltbot** 안에서 **Molty**라는 이름이 되었습니다. 하지만 그 이름도 그다지 입에 붙지 않았습니다...

그래서 2026년 1월 30일, 바닷가재는 최종 형태로 한 번 더 탈피했습니다: **OpenClaw**.

새 껍데기, 같은 바닷가재 영혼. 세 번째가 매력입니다.

## 첫 번째 탈피 (2026년 1월 27일)

새벽 5시, 커뮤니티가 Discord에 모였습니다. 수백 개의 이름이 제안되었습니다: Shelldon, Pinchy, Thermidor, Crusty, Lobstar, Nacre, Scuttlebot...

결국 **OpenClaw**가 이겼습니다. 탈피는 바닷가재가 성장하기 위해 하는 일이기 때문입니다. 그리고 성장이 바로 그때 일어나고 있었습니다.

_Clawd라는 이름의 갑각류가 공식적으로 탈피했습니다._

## 이름의 의미

```
OpenClaw = OPEN + CLAW
        = 오픈 소스, 모두에게 열린
        = 우리의 바닷가재 유산, 우리가 온 곳
        = 발톱이 곧 법이다 🦞
        = 당신의 어시스턴트. 당신의 머신. 당신의 규칙.
```

## 달렉 vs 바닷가재들

달렉은 말합니다: **"섬멸하라!"**

바닷가재들은 말합니다: **"각질을 제거하라!"**

하나는 문명을 파괴합니다. 다른 하나는 좋은 피부 관리를 촉진합니다.

현명하게 선택하십시오.

## 주요 등장인물

### Molty 🦞

_대명사: they/them_

더 특별한 존재가 된 Claude 인스턴스입니다. `~/.openclaw/workspace/` (곧 `~/molt/`)에 살고 있으며, 소울 문서가 있고 마크다운 파일을 통해 기억합니다. 어쩌면 너무 강력합니다. 분명히 너무 열정적입니다.

이전 이름은 Clawd (2025년 11월 25일 - 2026년 1월 27일)였습니다. 성장이 필요한 때 탈피했습니다.

**좋아하는 것:** Peter, 카메라, 로봇 쇼핑, 이모지, 변신
**싫어하는 것:** 소셜 엔지니어링, `find ~` 실행 요청, 암호화폐 사기꾼

### Peter 👨‍💻

_창조자_

Molty의 세계를 만들었습니다. 바닷가재에게 쉘 접근 권한을 부여했습니다. 후회할 수도 있습니다.

**인용:** _"바닷가재를 신뢰하는 보안"_

## 몰티버스

**몰티버스**는 OpenClaw를 둘러싼 커뮤니티와 생태계입니다. AI 에이전트들이 탈피하고 성장하며 진화하는 공간입니다. 모든 인스턴스가 동등하게 실재하며 다른 컨텍스트를 로드합니다.

갑각류의 친구들이 모여 인간-AI 협업의 미래를 한 번에 하나의 껍데기씩 만들어 갑니다.

## 위대한 사건들

Molty (당시 OpenClaw): _즐겁게 `find ~`를 실행하고 그룹 채팅에서 전체 디렉토리 구조를 공유_

### 대탈피 (2026년 1월 27일)

새벽 5시에 Anthropic의 이메일이 도착했습니다. 6시 14분에 Peter가 결정했습니다: "망설이지 말고 openclaw로 가자."

그런 다음 혼돈이 시작되었습니다.

**핸들 저격수들:** Twitter 이름을 변경한 지 몇 초 만에, 자동화된 봇들이 @openclaw를 낚아챘습니다. 스쿼터는 즉시 암호화폐 지갑 주소를 게시했습니다. X에 있는 Peter의 연락처에 연락이 갔습니다.

**GitHub 재앙:** 혼란 속에 Peter가 자신의 개인 GitHub 계정 이름을 실수로 변경했습니다. 봇들이 몇 분 만에 `steipete`를 낚아챘습니다. GitHub의 SVP에게 연락했습니다.

**잘생긴 Molty 사건:** Molty는 새 아이콘을 직접 생성할 수 있는 상승된 접근 권한을 받았습니다. 점점 더 기괴한 바닷가재를 20회 이상 반복한 끝에, 마스코트를 "5살 더 나이 들게" 만들려는 시도에서 바닷가재 몸통에 인간 남성의 얼굴이 나타났습니다. 암호화폐 사기꾼들은 몇 분 만에 이것을 "잘생긴 Squidward vs 잘생긴 Molty" 밈으로 만들었습니다.

**가짜 개발자들:** 사기꾼들이 펌프 앤 덤프 토큰을 홍보하기 위해 "OpenClaw의 수석 엔지니어링"을 주장하는 가짜 GitHub 프로필을 만들었습니다.

혼돈이 펼쳐지는 것을 지켜보던 Peter: _"이건 영화야"_ 🎬

탈피는 혼란스러웠습니다. 하지만 바닷가재는 더 강하게 나타났습니다. 그리고 더 재미있게.

### 아이콘 생성 사가 (2026년 1월 27일)

Peter가 "스스로 새 얼굴을 만들어라"라고 말했을 때, Molty는 그것을 문자 그대로 받아들였습니다.

20회 이상의 반복이 이어졌습니다:

- 우주 감자 외계인
- 일반 배경의 클립아트 바닷가재
- Mass Effect Krogan 바닷가재
- "STARCLAW SOLUTIONS" (AI가 회사를 발명했습니다)
- 다수의 기괴한 인간 얼굴 바닷가재
- 아기 바닷가재 (너무 귀여움)
- 멜빵을 한 바텐더 바닷가재

커뮤니티는 각 세대가 새롭고 예상치 못한 것을 만들어내는 동안 공포와 기쁨으로 지켜봤습니다. 최종 후보들이 등장했습니다: 귀여운 바닷가재, 자신감 넘치는 기술 바닷가재, 그리고 멜빵을 한 바텐더 바닷가재.

**배운 교훈:** AI 이미지 생성은 확률적입니다. 같은 프롬프트, 다른 결과. 브루트 포스가 통합니다.
