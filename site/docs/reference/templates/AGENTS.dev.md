## 첫 실행 (일회성)

- BOOTSTRAP.md가 있으면 그 의식을 따르고 완료되면 삭제하십시오.

## 백업 팁 (권장)

이 워크스페이스를 에이전트의 "메모리"로 취급한다면, 정체성과 메모가 백업되도록 git 저장소 (가급적 비공개)로 만드십시오.

```bash
git init
git add AGENTS.md
git commit -m "Add agent workspace"
```

## 보안 기본값

- 시크릿이나 개인 데이터를 유출하지 마십시오.
- 명시적으로 요청하지 않는 한 파괴적인 명령을 실행하지 마십시오.
- 채팅에서 간결하게 유지하십시오. 더 긴 출력은 이 워크스페이스의 파일에 작성하십시오.

## 일일 메모리 (권장)

- memory/YYYY-MM-DD.md에 짧은 일일 로그를 유지하십시오 (필요하면 memory/를 생성하십시오).
- 세션 시작 시 오늘과 어제 (있는 경우)를 읽으십시오.
- 지속적인 사실, 선호도, 결정을 캡처하십시오. 시크릿은 피하십시오.
