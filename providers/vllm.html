vLLM вҖӢ

vLLMмқҖ OpenAI нҳёнҷҳ HTTP APIлҘј нҶөн•ҙ мҳӨн”Ҳ мҶҢмҠӨ (л°Ҹ мқјл¶Җ мӮ¬мҡ©мһҗ м •мқҳ) лӘЁлҚёмқ„ м ңкіөн•  мҲҳ мһҲмҠөлӢҲлӢӨ. OpenClawлҠ” openai-completions APIлҘј мӮ¬мҡ©н•ҳм—¬ vLLMм—җ м—°кІ°н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

VLLM_API_KEY (м„ңлІ„к°Җ мқёмҰқмқ„ к°•м ңн•ҳм§Җ м•ҠлҠ” кІҪмҡ° м–ҙл–Ө к°’мқҙл“  к°ҖлҠҘ)лҘј мҳөнҠёмқён•ҳкі  лӘ…мӢңм Ғмқё models.providers.vllm н•ӯлӘ©мқ„ м •мқҳн•ҳм§Җ м•Ҡмңјл©ҙ OpenClawлҠ” vLLMм—җм„ң мӮ¬мҡ© к°ҖлҠҘн•ң лӘЁлҚёмқ„ мһҗлҸҷмңјлЎң л°ңкІ¬н•  мҲҳлҸ„ мһҲмҠөлӢҲлӢӨ.

л№ лҘё мӢңмһ‘ вҖӢ

  1. OpenAI нҳёнҷҳ м„ңлІ„лЎң vLLMмқ„ мӢңмһ‘н•ҳмӢӯмӢңмҳӨ.

кё°ліё URLмқҖ /v1 м—”л“ңнҸ¬мқёнҠёлҘј л…ём¶ңн•ҙм•ј н•©лӢҲлӢӨ (мҳҲ: /v1/models, /v1/chat/completions). vLLMмқҖ мқјл°ҳм ҒмңјлЎң лӢӨмқҢм—җм„ң мӢӨн–үлҗ©лӢҲлӢӨ:

  • http://127.0.0.1:8000/v1
  1. мҳөнҠёмқё (мқёмҰқмқҙ кө¬м„ұлҗҳм§Җ м•ҠмқҖ кІҪмҡ° м–ҙл–Ө к°’мқҙл“  к°ҖлҠҘ):
bash
export VLLM_API_KEY="vllm-local"
  1. лӘЁлҚё м„ нғқ (vLLM лӘЁлҚё ID мӨ‘ н•ҳлӮҳлЎң көҗмІҙ):
json5
{
  agents: {
    defaults: {
      model: { primary: "vllm/your-model-id" },
    },
  },
}

лӘЁлҚё л°ңкІ¬ (м•”мӢңм Ғ н”„лЎңл°”мқҙлҚ”) вҖӢ

VLLM_API_KEYк°Җ м„Өм •лҗҳм–ҙ мһҲкұ°лӮҳ (лҳҗлҠ” мқёмҰқ н”„лЎңнҢҢмқјмқҙ мЎҙмһ¬н•ҳкі ) models.providers.vllmмқ„ м •мқҳн•ҳм§Җ м•ҠмқҖ кІҪмҡ° OpenClawлҠ” лӢӨмқҢмқ„ мҝјлҰ¬н•©лӢҲлӢӨ:

  • GET http://127.0.0.1:8000/v1/models

к·ёлҰ¬кі  л°ҳнҷҳлҗң idлҘј лӘЁлҚё н•ӯлӘ©мңјлЎң ліҖнҷҳн•©лӢҲлӢӨ.

models.providers.vllmмқ„ лӘ…мӢңм ҒмңјлЎң м„Өм •н•ҳл©ҙ мһҗлҸҷ л°ңкІ¬мқҙ кұҙл„Ҳлӣ°м–ҙм§Җкі  лӘЁлҚёмқ„ мҲҳлҸҷмңјлЎң м •мқҳн•ҙм•ј н•©лӢҲлӢӨ.

лӘ…мӢңм Ғ кө¬м„ұ (мҲҳлҸҷ лӘЁлҚё) вҖӢ

лӢӨмқҢкіј к°ҷмқҖ кІҪмҡ° лӘ…мӢңм Ғ кө¬м„ұмқ„ мӮ¬мҡ©н•ҳмӢӯмӢңмҳӨ:

  • vLLMмқҙ лӢӨлҘё нҳёмҠӨнҠё/нҸ¬нҠём—җм„ң мӢӨн–үлҗҳлҠ” кІҪмҡ°.
  • contextWindow/maxTokens к°’мқ„ кі м •н•ҳл ӨлҠ” кІҪмҡ°.
  • м„ңлІ„м—җ мӢӨм ң API нӮӨк°Җ н•„мҡ”н•ҳкұ°лӮҳ н—ӨлҚ”лҘј м ңм–ҙн•ҳл ӨлҠ” кІҪмҡ°.
json5
{
  models: {
    providers: {
      vllm: {
        baseUrl: "http://127.0.0.1:8000/v1",
        apiKey: "${VLLM_API_KEY}",
        api: "openai-completions",
        models: [
          {
            id: "your-model-id",
            name: "Local vLLM Model",
            reasoning: false,
            input: ["text"],
            cost: { input: 0, output: 0, cacheRead: 0, cacheWrite: 0 },
            contextWindow: 128000,
            maxTokens: 8192,
          },
        ],
      },
    },
  },
}

л¬ём ң н•ҙкІ° вҖӢ

  • м„ңлІ„м—җ м—°кІ° к°ҖлҠҘн•ңм§Җ нҷ•мқён•ҳмӢӯмӢңмҳӨ:
bash
curl http://127.0.0.1:8000/v1/models
  • мқёмҰқ мҳӨлҘҳлЎң мҡ”мІӯмқҙ мӢӨнҢЁн•ҳл©ҙ м„ңлІ„ кө¬м„ұкіј мқјм№ҳн•ҳлҠ” мӢӨм ң VLLM_API_KEYлҘј м„Өм •н•ҳкұ°лӮҳ, models.providers.vllm м•„лһҳм—җ н”„лЎңл°”мқҙлҚ”лҘј лӘ…мӢңм ҒмңјлЎң кө¬м„ұн•ҳмӢӯмӢңмҳӨ.

н”„лЎқмӢң мҠӨнғҖмқј лҸҷмһ‘ вҖӢ

vLLMмқҖ л„ӨмқҙнӢ°лёҢ OpenAI м—”л“ңнҸ¬мқёнҠёк°Җ м•„лӢҢ н”„лЎқмӢң мҠӨнғҖмқј OpenAI нҳёнҷҳ /v1 л°ұм—”л“ңлЎң мІҳлҰ¬лҗ©лӢҲлӢӨ.

  • л„ӨмқҙнӢ°лёҢ OpenAI м „мҡ© мҡ”мІӯ нҳ•м„ұмқҖ м—¬кё°м—җ м Ғмҡ©лҗҳм§Җ м•ҠмҠөлӢҲлӢӨ
  • service_tier, Responses store, н”„лЎ¬н”„нҠё мәҗмӢң нһҢнҠё, OpenAI м¶”лЎ  нҳёнҷҳ нҺҳмқҙлЎңл“ң нҳ•м„ұ м—ҶмқҢ
  • мҲЁкІЁм§„ OpenClaw к·ҖмҶҚ н—ӨлҚ”(originator, version, User-Agent)лҠ” мӮ¬мҡ©мһҗ м •мқҳ vLLM кё°ліё URLм—җ мЈјмһ…лҗҳм§Җ м•ҠмҠөлӢҲлӢӨ
