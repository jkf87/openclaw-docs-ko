CLI м„Өм • м°ёмЎ° вҖӢ

мқҙ нҺҳмқҙм§ҖлҠ” openclaw onboardмқҳ м „мІҙ м°ёмЎ°мһ…лӢҲлӢӨ. к°„лӢЁн•ң к°Җмқҙл“ңлҠ” мҳЁліҙл”© (CLI)лҘј м°ёмЎ°н•ҳмӢӯмӢңмҳӨ.

л§ҲлІ•мӮ¬к°Җ мҲҳн–үн•ҳлҠ” мһ‘м—… вҖӢ

лЎңм»¬ лӘЁл“ң (кё°ліёк°’)лҠ” лӢӨмқҢмқ„ м•ҲлӮҙн•©лӢҲлӢӨ:

  • лӘЁлҚё л°Ҹ мқёмҰқ м„Өм • (OpenAI Code кө¬лҸ… OAuth, Anthropic Claude CLI лҳҗлҠ” API нӮӨ, MiniMax, GLM, Ollama, Moonshot, StepFun л°Ҹ AI Gateway мҳөм…ҳ нҸ¬н•Ё)
  • мӣҢнҒ¬мҠӨнҺҳмқҙмҠӨ мң„м№ҳ л°Ҹ л¶ҖнҠёмҠӨнҠёлһ© нҢҢмқј
  • Gateway м„Өм • (нҸ¬нҠё, л°”мқёл“ң, мқёмҰқ, tailscale)
  • мұ„л„җ л°Ҹ н”„лЎңл°”мқҙлҚ” (Telegram, WhatsApp, Discord, Google Chat, Mattermost, Signal, BlueBubbles л°Ҹ кё°нғҖ лІҲл“Ө мұ„л„җ н”Ңлҹ¬к·ёмқё)
  • лҚ°лӘ¬ м„Өм№ҳ (LaunchAgent, systemd мӮ¬мҡ©мһҗ мң лӢӣ лҳҗлҠ” Startup нҸҙлҚ” лҢҖмІҙ кё°лҠҘмқҙ мһҲлҠ” л„ӨмқҙнӢ°лёҢ Windows мҳҲм•Ҫ мһ‘м—…)
  • н—¬мҠӨ мІҙнҒ¬
  • мҠӨнӮ¬ м„Өм •

мӣҗкІ© лӘЁл“ңлҠ” мқҙ лЁёмӢ мқҙ лӢӨлҘё кіімқҳ кІҢмқҙнҠёмӣЁмқҙм—җ м—°кІ°н•ҳлҸ„лЎқ кө¬м„ұн•©лӢҲлӢӨ. мӣҗкІ© нҳёмҠӨнҠём—җм„ңлҠ” м•„л¬ҙкІғлҸ„ м„Өм№ҳн•ҳкұ°лӮҳ мҲҳм •н•ҳм§Җ м•ҠмҠөлӢҲлӢӨ.

лЎңм»¬ нқҗлҰ„ м„ёл¶Җ м •ліҙ вҖӢ

  1. кё°мЎҙ кө¬м„ұ к°җм§Җ

    • ~/.openclaw/openclaw.jsonмқҙ мЎҙмһ¬н•ҳл©ҙ мң м§Җ, мҲҳм • лҳҗлҠ” мһ¬м„Өм •мқ„ м„ нғқн•©лӢҲлӢӨ.
      • л§ҲлІ•мӮ¬лҘј лӢӨмӢң мӢӨн–үн•ҙлҸ„ мһ¬м„Өм •мқ„ лӘ…мӢңм ҒмңјлЎң м„ нғқн•ҳм§Җ м•ҠлҠ” н•ң (лҳҗлҠ” --resetмқ„ м „лӢ¬н•ҳм§Җ м•ҠлҠ” н•ң) м•„л¬ҙкІғлҸ„ м§Җмҡ°м§Җ м•ҠмҠөлӢҲлӢӨ.
      • CLI --resetмқҖ кё°ліёк°’мңјлЎң config+creds+sessionsмһ…лӢҲлӢӨ; мӣҢнҒ¬мҠӨнҺҳмқҙмҠӨлҸ„ м ңкұ°н•ҳл Өл©ҙ --reset-scope fullмқ„ мӮ¬мҡ©н•ҳмӢӯмӢңмҳӨ.
      • кө¬м„ұмқҙ мһҳлӘ»лҗҳм—Ҳкұ°лӮҳ л Ҳкұ°мӢң нӮӨк°Җ нҸ¬н•Ёлҗң кІҪмҡ°, л§ҲлІ•мӮ¬лҠ” кі„мҶҚн•ҳкё° м „м—җ openclaw doctorлҘј мӢӨн–үн•ҳлҸ„лЎқ мҡ”мІӯн•©лӢҲлӢӨ.
      • мһ¬м„Өм •мқҖ trashлҘј мӮ¬мҡ©н•ҳкі  лӢӨмқҢ лІ”мң„лҘј м ңкіөн•©лӢҲлӢӨ:
        • кө¬м„ұл§Ң
        • кө¬м„ұ + мһҗкІ© мҰқлӘ… + м„ём…ҳ
        • м „мІҙ мһ¬м„Өм • (мӣҢнҒ¬мҠӨнҺҳмқҙмҠӨлҸ„ м ңкұ°)

  2. лӘЁлҚё л°Ҹ мқёмҰқ

  • м „мІҙ мҳөм…ҳ л§ӨнҠёлҰӯмҠӨлҠ” мқёмҰқ л°Ҹ лӘЁлҚё мҳөм…ҳм—җ мһҲмҠөлӢҲлӢӨ.
  1. мӣҢнҒ¬мҠӨнҺҳмқҙмҠӨ
  • кё°ліёк°’ ~/.openclaw/workspace (кө¬м„ұ к°ҖлҠҘ).
    • мөңмҙҲ мӢӨн–ү л¶ҖнҠёмҠӨнҠёлһ© мқҳмӢқм—җ н•„мҡ”н•ң мӣҢнҒ¬мҠӨнҺҳмқҙмҠӨ нҢҢмқјмқ„ мҙҲкё°нҷ”н•©лӢҲлӢӨ.
    • мӣҢнҒ¬мҠӨнҺҳмқҙмҠӨ л Ҳмқҙм•„мӣғ: м—җмқҙм „нҠё мӣҢнҒ¬мҠӨнҺҳмқҙмҠӨ.
  1. Gateway
  • нҸ¬нҠё, л°”мқёл“ң, мқёмҰқ лӘЁл“ң л°Ҹ tailscale л…ём¶ңм—җ лҢҖн•ң н”„лЎ¬н”„нҠё.
    • к¶ҢмһҘ: лЎңм»¬ WS нҒҙлқјмқҙм–ёнҠёк°Җ мқёмҰқн•ҙм•ј н•ҳлҸ„лЎқ лЈЁн”„л°ұм—җм„ңлҸ„ нҶ нҒ° мқёмҰқмқ„ нҷңм„ұнҷ”н•ң мғҒнғңлЎң мң м§Җн•ҳмӢӯмӢңмҳӨ.
    • нҶ нҒ° лӘЁл“ңм—җм„ң, лҢҖнҷ”нҳ• м„Өм •мқҖ лӢӨмқҢмқ„ м ңкіөн•©лӢҲлӢӨ:
      • нҸүл¬ё нҶ нҒ° мғқм„ұ/м ҖмһҘ (кё°ліёк°’)
      • SecretRef мӮ¬мҡ© (мҳөнҠёмқё)
    • л№„л°ҖлІҲнҳё лӘЁл“ңм—җм„ң, лҢҖнҷ”нҳ• м„Өм •лҸ„ нҸүл¬ё лҳҗлҠ” SecretRef м ҖмһҘмқ„ м§Җмӣҗн•©лӢҲлӢӨ.
    • л№„лҢҖнҷ”нҳ• нҶ нҒ° SecretRef кІҪлЎң: --gateway-token-ref-env &lt;ENV_VAR&gt;.
      • мҳЁліҙл”© н”„лЎңм„ёмҠӨ нҷҳкІҪм—җм„ң л№„м–ҙ мһҲм§Җ м•ҠмқҖ нҷҳкІҪ ліҖмҲҳк°Җ н•„мҡ”н•©лӢҲлӢӨ.
      • --gateway-tokenкіј кІ°н•©н•  мҲҳ м—ҶмҠөлӢҲлӢӨ.
    • лӘЁл“  лЎңм»¬ н”„лЎңм„ёмҠӨлҘј мҷ„м „нһҲ мӢ лў°н•ҳлҠ” кІҪмҡ°м—җл§Ң мқёмҰқмқ„ л№„нҷңм„ұнҷ”н•ҳмӢӯмӢңмҳӨ.
    • лЈЁн”„л°ұмқҙ м•„лӢҢ л°”мқёл“ңм—җлҠ” м—¬м „нһҲ мқёмҰқмқҙ н•„мҡ”н•©лӢҲлӢӨ.
  1. мұ„л„җ
  • WhatsApp: м„ нғқм Ғ QR лЎңк·ёмқё
    • Telegram: лҙҮ нҶ нҒ°
    • Discord: лҙҮ нҶ нҒ°
    • Google Chat: м„ңл№„мҠӨ кі„м • JSON + мӣ№нӣ… лҢҖмғҒ
    • Mattermost: лҙҮ нҶ нҒ° + кё°ліё URL
    • Signal: м„ нғқм Ғ signal-cli м„Өм№ҳ + кі„м • кө¬м„ұ
    • BlueBubbles: iMessageм—җ к¶ҢмһҘ; м„ңлІ„ URL + л№„л°ҖлІҲнҳё + мӣ№нӣ…
    • iMessage: л Ҳкұ°мӢң imsg CLI кІҪлЎң + DB м ‘к·ј
    • DM ліҙм•Ҳ: кё°ліёк°’мқҖ нҺҳм–ҙл§Ғмһ…лӢҲлӢӨ. мІ« лІҲм§ё DMмқҖ мҪ”л“ңлҘј ліҙлғ…лӢҲлӢӨ; openclaw pairing approve &lt;channel&gt; <code>лҘј нҶөн•ҙ мҠ№мқён•ҳкұ°лӮҳ н—Ҳмҡ© лӘ©лЎқмқ„ мӮ¬мҡ©н•ҳмӢӯмӢңмҳӨ.
  1. лҚ°лӘ¬ м„Өм№ҳ
  • macOS: LaunchAgent - лЎңк·ёмқёлҗң мӮ¬мҡ©мһҗ м„ём…ҳмқҙ н•„мҡ”н•©лӢҲлӢӨ; н—Өл“ңлҰ¬мҠӨмқҳ кІҪмҡ° л§һм¶Ө LaunchDaemonмқ„ мӮ¬мҡ©н•ҳмӢӯмӢңмҳӨ (м ңкіөлҗҳм§Җ м•ҠмқҢ).
    • WSL2лҘј нҶөн•ң Linux л°Ҹ Windows: systemd мӮ¬мҡ©мһҗ мң лӢӣ
      • л§ҲлІ•мӮ¬лҠ” лЎңк·ём•„мӣғ нӣ„м—җлҸ„ кІҢмқҙнҠёмӣЁмқҙк°Җ кі„мҶҚ мӢӨн–үлҗҳлҸ„лЎқ loginctl enable-linger &lt;user&gt;лҘј мӢңлҸ„н•©лӢҲлӢӨ.
      • sudoлҘј мҡ”мІӯн•  мҲҳ мһҲмҠөлӢҲлӢӨ (/var/lib/systemd/lingerлҘј мһ‘м„ұн•©лӢҲлӢӨ); лЁјм Җ sudo м—Ҷмқҙ мӢңлҸ„н•©лӢҲлӢӨ.
    • л„ӨмқҙнӢ°лёҢ Windows: мҳҲм•Ҫ мһ‘м—… мҡ°м„ 
      • мһ‘м—… мғқм„ұмқҙ кұ°л¶Җлҗҳл©ҙ OpenClawлҠ” мӮ¬мҡ©мһҗлі„ Startup нҸҙлҚ” лЎңк·ёмқё н•ӯлӘ©мңјлЎң лҢҖмІҙн•ҳкі  мҰүмӢң кІҢмқҙнҠёмӣЁмқҙлҘј мӢңмһ‘н•©лӢҲлӢӨ.
      • лҚ” лӮҳмқҖ к°җлҸ… мғҒнғңлҘј м ңкіөн•ҳлҜҖлЎң мҳҲм•Ҫ мһ‘м—…мқҙ м„ нҳёлҗ©лӢҲлӢӨ.
    • лҹ°нғҖмһ„ м„ нғқ: Node (к¶ҢмһҘ; WhatsApp л°Ҹ Telegramм—җ н•„мҡ”). BunмқҖ к¶ҢмһҘн•ҳм§Җ м•ҠмҠөлӢҲлӢӨ.
  1. н—¬мҠӨ мІҙнҒ¬
  • кІҢмқҙнҠёмӣЁмқҙлҘј мӢңмһ‘н•ҳкі  (н•„мҡ”н•ң кІҪмҡ°) openclaw healthлҘј мӢӨн–үн•©лӢҲлӢӨ.
    • openclaw status --deepмқҖ мұ„л„җ н”„лЎңлёҢк°Җ м§Җмӣҗлҗ  л•Ң мұ„л„җ н”„лЎңлёҢлҘј нҸ¬н•Ён•ң лқјмқҙлёҢ кІҢмқҙнҠёмӣЁмқҙ н—¬мҠӨ н”„лЎңлёҢлҘј мғҒнғң м¶ңл Ҙм—җ м¶”к°Җн•©лӢҲлӢӨ.
  1. мҠӨнӮ¬
  • мӮ¬мҡ© к°ҖлҠҘн•ң мҠӨнӮ¬мқ„ мқҪкі  мҡ”кө¬ мӮ¬н•ӯмқ„ нҷ•мқён•©лӢҲлӢӨ.
    • л…ёл“ң кҙҖлҰ¬мһҗ м„ нғқ: npm, pnpm лҳҗлҠ” bun.
    • м„ нғқм Ғ мқҳмЎҙм„ұмқ„ м„Өм№ҳн•©лӢҲлӢӨ (мқјл¶ҖлҠ” macOSм—җм„ң HomebrewлҘј мӮ¬мҡ©н•©лӢҲлӢӨ).
  1. мҷ„лЈҢ
  • мҡ”м•Ҫ л°Ҹ iOS, Android л°Ҹ macOS м•ұ мҳөм…ҳмқ„ нҸ¬н•Ён•ң лӢӨмқҢ лӢЁкі„.

NOTE

GUIк°Җ к°җм§Җлҗҳм§Җ м•Ҡмңјл©ҙ, л§ҲлІ•мӮ¬лҠ” лёҢлқјмҡ°м ҖлҘј м—¬лҠ” лҢҖмӢ  Control UIм—җ лҢҖн•ң SSH нҸ¬нҠё нҸ¬мӣҢл“ң м§Җм№Ёмқ„ м¶ңл Ҙн•©лӢҲлӢӨ. Control UI м—җм…Ӣмқҙ м—Ҷмңјл©ҙ л§ҲлІ•мӮ¬к°Җ л№Ңл“ңлҘј мӢңлҸ„н•©лӢҲлӢӨ; лҢҖмІҙлҠ” pnpm ui:buildмһ…лӢҲлӢӨ (UI мқҳмЎҙм„ұ мһҗлҸҷ м„Өм№ҳ).

мӣҗкІ© лӘЁл“ң м„ёл¶Җ м •ліҙ вҖӢ

мӣҗкІ© лӘЁл“ңлҠ” мқҙ лЁёмӢ мқҙ лӢӨлҘё кіімқҳ кІҢмқҙнҠёмӣЁмқҙм—җ м—°кІ°н•ҳлҸ„лЎқ кө¬м„ұн•©лӢҲлӢӨ.

INFO

мӣҗкІ© лӘЁл“ңлҠ” мӣҗкІ© нҳёмҠӨнҠём—җм„ң м•„л¬ҙкІғлҸ„ м„Өм№ҳн•ҳкұ°лӮҳ мҲҳм •н•ҳм§Җ м•ҠмҠөлӢҲлӢӨ.

м„Өм • н•ӯлӘ©:

  • мӣҗкІ© кІҢмқҙнҠёмӣЁмқҙ URL (ws://...)
  • мӣҗкІ© кІҢмқҙнҠёмӣЁмқҙ мқёмҰқмқҙ н•„мҡ”н•ң кІҪмҡ° нҶ нҒ° (к¶ҢмһҘ)

NOTE

  • кІҢмқҙнҠёмӣЁмқҙк°Җ лЈЁн”„л°ұ м „мҡ©мқё кІҪмҡ° SSH н„°л„җл§Ғ лҳҗлҠ” tailnetмқ„ мӮ¬мҡ©н•ҳмӢӯмӢңмҳӨ.
  • л””мҠӨм»ӨлІ„лҰ¬ нһҢнҠё:
    • macOS: Bonjour (dns-sd)
    • Linux: Avahi (avahi-browse)

л§һм¶Өнҳ• н”„лЎңл°”мқҙлҚ”

лҢҖнҷ”нҳ• мҳЁліҙл”©мқҖ лӢӨлҘё н”„лЎңл°”мқҙлҚ” API нӮӨ нқҗлҰ„кіј лҸҷмқјн•ң API нӮӨ м ҖмһҘ м„ нғқмқ„ м§Җмӣҗн•©лӢҲлӢӨ:
- **м§ҖкёҲ API нӮӨ л¶ҷм—¬л„Јкё°** (нҸүл¬ё)
- **л№„л°Җ м°ёмЎ° мӮ¬мҡ©** (нҷҳкІҪ ліҖмҲҳ м°ёмЎ° лҳҗлҠ” кө¬м„ұлҗң н”„лЎңл°”мқҙлҚ” м°ёмЎ°, мӮ¬м „ кІҖмҰқ нҸ¬н•Ё)

л№„лҢҖнҷ”нҳ• н”Ңлһҳк·ё:
- `--auth-choice custom-api-key`
- `--custom-base-url`
- `--custom-model-id`
- `--custom-api-key` (м„ нғқ мӮ¬н•ӯ; `CUSTOM_API_KEY`лЎң лҢҖмІҙлҗЁ)
- `--custom-provider-id` (м„ нғқ мӮ¬н•ӯ)
- `--custom-compatibility &lt;openai|anthropic&gt;` (м„ нғқ мӮ¬н•ӯ; кё°ліёк°’ `openai`)
кұҙл„Ҳлӣ°кё°

мқёмҰқмқ„ кө¬м„ұлҗҳм§Җ м•ҠмқҖ мғҒнғңлЎң л‘ЎлӢҲлӢӨ.

лӘЁлҚё лҸҷмһ‘:

  • к°җм§Җлҗң мҳөм…ҳм—җм„ң кё°ліё лӘЁлҚёмқ„ м„ нғқн•ҳкұ°лӮҳ н”„лЎңл°”мқҙлҚ” л°Ҹ лӘЁлҚёмқ„ мҲҳлҸҷмңјлЎң мһ…л Ҙн•©лӢҲлӢӨ.
  • мҳЁліҙл”©мқҙ н”„лЎңл°”мқҙлҚ” мқёмҰқ м„ нғқм—җм„ң мӢңмһ‘лҗҳл©ҙ, лӘЁлҚё м„ нғқкё°лҠ” н•ҙлӢ№ н”„лЎңл°”мқҙлҚ”лҘј мһҗлҸҷмңјлЎң м„ нҳён•©лӢҲлӢӨ. Volcengine л°Ҹ BytePlusмқҳ кІҪмҡ°, лҸҷмқјн•ң м„ нҳёлҸ„к°Җ мҪ”л”© н”Ңлһң ліҖнҳ• (volcengine-plan/*, byteplus-plan/*)м—җлҸ„ м Ғмҡ©лҗ©лӢҲлӢӨ.
  • м„ нҳё н”„лЎңл°”мқҙлҚ” н•„н„°к°Җ л№„м–ҙ мһҲмңјл©ҙ, м„ нғқкё°лҠ” лӘЁлҚёмқҙ м—ҶлҠ” мғҒнғңлЎң н‘ңмӢңн•ҳлҠ” лҢҖмӢ  м „мІҙ м№ҙнғҲлЎңк·ёлЎң лҢҖмІҙлҗ©лӢҲлӢӨ.
  • л§ҲлІ•мӮ¬лҠ” лӘЁлҚё нҷ•мқёмқ„ мӢӨн–үн•ҳкі  кө¬м„ұлҗң лӘЁлҚёмқҙ м•Ң мҲҳ м—Ҷкұ°лӮҳ мқёмҰқмқҙ м—ҶлҠ” кІҪмҡ° кІҪкі н•©лӢҲлӢӨ.

мһҗкІ© мҰқлӘ… л°Ҹ н”„лЎңнҢҢмқј кІҪлЎң:

  • мқёмҰқ н”„лЎңнҢҢмқј (API нӮӨ + OAuth): ~/.openclaw/agents/&lt;agentId&gt;/agent/auth-profiles.json
  • л Ҳкұ°мӢң OAuth к°Җм ёмҳӨкё°: ~/.openclaw/credentials/oauth.json

мһҗкІ© мҰқлӘ… м ҖмһҘ лӘЁл“ң:

  • кё°ліё мҳЁліҙл”© лҸҷмһ‘мқҖ API нӮӨлҘј мқёмҰқ н”„лЎңнҢҢмқјмқҳ нҸүл¬ё к°’мңјлЎң мң м§Җн•©лӢҲлӢӨ.
  • --secret-input-mode refлҠ” нҸүл¬ё нӮӨ м ҖмһҘ лҢҖмӢ  м°ёмЎ° лӘЁл“ңлҘј нҷңм„ұнҷ”н•©лӢҲлӢӨ. лҢҖнҷ”нҳ• м„Өм •м—җм„ң лӢӨмқҢ мӨ‘ н•ҳлӮҳлҘј м„ нғқн•  мҲҳ мһҲмҠөлӢҲлӢӨ:
    • нҷҳкІҪ ліҖмҲҳ м°ёмЎ° (мҳҲ: keyRef: { source: "env", provider: "default", id: "OPENAI_API_KEY" })
    • кө¬м„ұлҗң н”„лЎңл°”мқҙлҚ” м°ёмЎ° (file лҳҗлҠ” exec) (н”„лЎңл°”мқҙлҚ” лі„м№ӯ + id нҸ¬н•Ё)
  • лҢҖнҷ”нҳ• м°ёмЎ° лӘЁл“ңлҠ” м ҖмһҘ м „ л№ лҘё мӮ¬м „ кІҖмҰқмқ„ мӢӨн–үн•©лӢҲлӢӨ.
    • нҷҳкІҪ ліҖмҲҳ м°ёмЎ°: нҳ„мһ¬ мҳЁліҙл”© нҷҳкІҪм—җм„ң ліҖмҲҳ мқҙлҰ„ + л№„м–ҙ мһҲм§Җ м•ҠмқҖ к°’мқ„ кІҖмҰқн•©лӢҲлӢӨ.
    • н”„лЎңл°”мқҙлҚ” м°ёмЎ°: н”„лЎңл°”мқҙлҚ” кө¬м„ұмқ„ кІҖмҰқн•ҳкі  мҡ”мІӯлҗң idлҘј нҷ•мқён•©лӢҲлӢӨ.
    • мӮ¬м „ кІҖмҰқмқҙ мӢӨнҢЁн•ҳл©ҙ мҳЁліҙл”©мқҙ мҳӨлҘҳлҘј н‘ңмӢңн•ҳкі  мһ¬мӢңлҸ„н•  мҲҳ мһҲмҠөлӢҲлӢӨ.
  • л№„лҢҖнҷ”нҳ• лӘЁл“ңм—җм„ң --secret-input-mode refлҠ” нҷҳкІҪ ліҖмҲҳ кё°л°ҳл§Ң м§Җмӣҗн•©лӢҲлӢӨ.
    • мҳЁліҙл”© н”„лЎңм„ёмҠӨ нҷҳкІҪм—җм„ң н”„лЎңл°”мқҙлҚ” нҷҳкІҪ ліҖмҲҳлҘј м„Өм •н•ҳмӢӯмӢңмҳӨ.
    • мқёлқјмқё нӮӨ н”Ңлһҳк·ё (мҳҲ: --openai-api-key)лҠ” н•ҙлӢ№ нҷҳкІҪ ліҖмҲҳк°Җ м„Өм •лҗҳм–ҙ мһҲм–ҙм•ј н•©лӢҲлӢӨ; к·ёл Үм§Җ м•Ҡмңјл©ҙ мҳЁліҙл”©мқҙ л№ лҘҙкІҢ мӢӨнҢЁн•©лӢҲлӢӨ.
