---
summary: "openclaw security audit가 발생시키는 checkId 참조 카탈로그"
read_when:
  - `openclaw security audit` 출력에서 특정 `checkId`를 보았고 그 의미를 알고 싶을 때
  - 주어진 finding에 대한 fix 키/경로가 필요할 때
  - security audit 실행 전반에 걸쳐 심각도를 분류하고 있을 때
title: "Security audit 체크"
---

`openclaw security audit`는 `checkId`로 키가 지정된 구조화된 finding을 방출합니다. 이
페이지는 해당 ID에 대한 참조 카탈로그입니다. 상위 수준의 위협 모델과
하드닝 가이드는 [Security](/gateway/security/)를 참조하세요.

실제 배포에서 가장 자주 보게 될 높은 신호의 `checkId` 값들입니다 (전체가
아닙니다):
