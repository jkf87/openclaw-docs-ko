# TOOLS.md - 로컬 메모

스킬은 도구가 _어떻게_ 작동하는지 정의합니다. 이 파일은 _당신의_ 구체적인 것 — 설정에 고유한 것들을 위한 것입니다.

## 여기에 들어가는 것

다음과 같은 것들:

- 카메라 이름 및 위치
- SSH 호스트 및 별칭
- TTS를 위한 선호 음성
- 스피커/룸 이름
- 기기 별칭
- 환경별 모든 것

## 예시

```markdown
### 카메라

- living-room → 메인 구역, 180° 광각
- front-door → 입구, 모션 트리거

### SSH

- home-server → 192.168.1.100, 사용자: admin

### TTS

- 선호 음성: "Nova" (따뜻하고, 약간 영국식)
- 기본 스피커: 주방 HomePod
```

## 왜 별도로?

스킬은 공유됩니다. 당신의 설정은 당신의 것입니다. 분리하면 메모를 잃지 않고 스킬을 업데이트할 수 있고, 인프라를 누출하지 않고 스킬을 공유할 수 있습니다.
