# `openclaw tasks flow`

Flow 명령은 독립 실행형 `flows` 명령이 아닌 `openclaw tasks`의 하위 명령입니다.

```bash
openclaw tasks flow list [--json]
openclaw tasks flow show &lt;lookup&gt;
openclaw tasks flow cancel &lt;lookup&gt;
```

전체 문서는 [태스크 Flow](/automation/taskflow) 및 [tasks CLI 참조](/cli/index#tasks)를 참조하세요.
