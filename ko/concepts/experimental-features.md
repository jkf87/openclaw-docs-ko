# 실험적 기능

OpenClaw의 실험적 기능은 **opt-in 프리뷰 서페이스**입니다. 안정적인 기본값이나 장기적인 공개 계약이 될 자격을 갖추기 전에 현실 세계에서 더 많이 검증되어야 하기 때문에 명시적인 플래그 뒤에 둡니다.

일반 설정과는 다르게 취급하세요:

- 관련 문서가 시도를 권하지 않는 한 **기본적으로 끄세요**.
- 안정 설정보다 **형태와 동작이 더 빠르게 변경**될 수 있다고 예상하세요.
- 안정 경로가 이미 존재하면 그것을 먼저 사용하세요.
- OpenClaw를 광범위하게 배포 중이라면, 공유 기준에 포함하기 전에 더 작은 환경에서 실험 플래그를 테스트하세요.

## 현재 문서화된 플래그

| 서페이스              | 키                                                        | 언제 사용하는가                                                                                                  | 자세히                                                                                                    |
| --------------------- | --------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------- |
| 로컬 모델 런타임      | `agents.defaults.experimental.localModelLean`             | 더 작거나 엄격한 로컬 백엔드가 OpenClaw의 전체 기본 도구 서페이스에서 막힐 때                                    | [로컬 모델](/gateway/local-models)                                                                        |
| 메모리 검색           | `agents.defaults.memorySearch.experimental.sessionMemory` | `memory_search`가 이전 세션 기록을 인덱스하길 원하고 추가 저장/인덱싱 비용을 감수할 수 있을 때                   | [메모리 설정 참조](/reference/memory-config#session-memory-search-experimental)                           |
| 구조화된 계획 도구    | `tools.experimental.planTool`                             | 호환되는 런타임과 UI에서 다단계 작업 추적을 위한 구조화된 `update_plan` 도구를 노출하려 할 때                    | [Gateway 설정 참조](/gateway/configuration-reference#toolsexperimental)                                   |

## 로컬 모델 린(lean) 모드

`agents.defaults.experimental.localModelLean: true`는 성능이 떨어지는 로컬 모델 환경을 위한 압력 해소 밸브입니다. `browser`, `cron`, `message` 같은 무거운 기본 도구들을 쳐내, 작은 컨텍스트나 더 엄격한 OpenAI 호환 백엔드에서 프롬프트 형태가 더 작고 덜 취약해지도록 합니다.

이는 의도적으로 **일반 경로가 아닙니다**. 백엔드가 전체 런타임을 깔끔하게 처리한다면 이 플래그는 꺼두세요.

## 실험적 = 숨겨져 있음이 아니다

어떤 기능이 실험적이라면 OpenClaw는 문서와 설정 경로 자체에서 그것을 명확히 표기해야 합니다. 프리뷰 동작을 안정적으로 보이는 기본 손잡이 안에 슬쩍 집어넣고 정상인 척해서는 **안 됩니다**. 그렇게 하면 설정 서페이스가 지저분해지기 시작합니다.
